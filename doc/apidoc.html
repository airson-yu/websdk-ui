调度台WEB接口说明文档

一.文件结构

调度台WEB接口包含的文件：

1.dist目录：websdk资源目录。

1-1.dist/websdk.bundle.js：包含了所有的接口功能。

1-2.dist/assets目录：存放图片、字体、音频等资源文件。

2.doc目录：API文档。

3.demo目录：演示案例。

二.集成方式

安装调度台软件：

请先确保已下载并成功安装调度台服务软件（ConsoleService-8.0.37.exe）：（链接: https://pan.baidu.com/s/1ZYXOMzWDFLkZMOjKwYiRPg 密码: puno） 点击下载

前端集成：

1.将websdk.bundle.js，assets复制到首页的同级目录,assets须与首页保持同一目录，websdk.bundle.js非必须。

2.在首页中引入websdk.bundle.js

3.在JS中调用对应的API,具体查看demo.html,demo.js

注意事项：

1.若在调用登陆接口时，返回的cmd_status为5，则使用admin权限执行这两个命令：1.net stop "ITC Console Service" 2.net start "ITC Console Service"

2.改了配置文件，需要手动重启服务，服务名为：集群调度服务

3.logon接口新增了三个参数：ipaddr, port, orgid

4.视频相关的请求和监听函数移除了camera参数

5.在IE浏览器中禁止提出提示框（已限制此网页运行脚本或ActiveX控件），只需要在网页的html标签与head标签之间加上如下内容：

<!-- saved from url=(0016)http://localhost -->

三.API整体结构

    websdk:
        request:
            authRequest:
                logon:登陆
                logout:退出登陆
                emergencyHandled:告警已处理
                changePasswd:修改密码
            userRequest:
                getUserInfo:获取用户信息
                getConsoleInfo:获取调度台信息
                setUserParams:设置用户参数
                getUserStateAsync:获取用户状态（通过userStateNotice异步通知）老版本的noticeUserState接口仍保留
                addAdminUsers:批量管辖用户
            gpsRequest:
                queryGPS:查询终端的当前位置
                queryHistoryGPS:查询终端的GPS轨迹
            groupRequest:
                getGroupInfo:获取群组信息
                getGroupAttachInfo:获取群组成员信息和进入群组成员信息
                enterGroup:加入群组
                leaveGroup:退出群组
                forceEnterGroup:群组强拉（强行要求组的成员进入该组）
                forceLeaveGroup:群组强拆（强行解散该组，要求当前正在该组的成员立即回到被强拖时所在的组）
                addGroupMember:增加组成员
                removeGroupMember:移除组成员
                createGroup:创建临时组
                deleteGroup:删除临时组
            imRequest:
                sendIM:发送消息
            videoRequest:
                playVideo:播放视频
                stopVideo:停止视频
                switchCamera:切换摄像头
                updateVideoSet:更改分辨率或视频质量
                getVideoList:获取视频列表
                transformVideo:转换视频格式（mp4转m3u8）
                getPushVideoUsers:获取正在视频推流的用户列表
                setPushVideoPlayType:设置主动推流的视频是否返回URL
            
            voiceRequest:
                call:请求语音通话
                callStatus:响应语音通话
                pttOn:开始PTT
                pttOff:结束PTT
                dtmf:PSTN呼叫分机号（暂不支持）
                getAudioList:获取语音列表
                transformAudio:转换语音格式（amr转mp3）
        view:
            showUserModal:显示用户窗口
            showGroupModal:显示群组窗口
            showCreateGroupModal:显示创建临时组窗口
            getPlayVideoUids:获取正在视频推流并播放的用户ID列表

        config:
            set_video_close_action:设置关闭视频时的操作（包括终端主动推送和调度台拉取）
            set_video_pull_close_action:设置关闭视频时的操作（只针对主动推送的视频）
            set_video_push_close_action:设置关闭视频时的操作（只针对调度台拉取的视频）
            setLogLevelDebug:日志打印级别设置为none(不打印)
            setLogLevelDebug:日志打印级别设置为debug

        listeners:
            logonNotice:登录状态变更通知
            logoutNotice:退出登录通知
            emergencyAlarmNotice:告警通知
            emergencyHandledNotice:告警已处理通知
            userProfileNotice:用户信息通知
            userParamsNotice:用户配置参数通知
            userStateNotice:用户状态通知
            userGPSNotice:用户GPS通知
            callStatusNotice:语音通话状态状态通知
            pttStatusNotice:PTT状态通知
            groupAttachInfoNotice:群组成员数和进入群组成员数的结果通知
            enterGroupNotice:加入群组通知
            leaveGroupNotice:离开群组通知
            addGroupMemberNotice:增加组成员通知
            removeGroupMemberNotice:移除组成员通知
            groupMemStatusNotice:组成员状态通知
            imNotice:IM消息通知
            playVideoNotice:打开视频通知
            stopPlayVideoNotice:关闭视频通知
            playVideoRspNotice:打开视频的结果通知
            stopPlayVideoRspNotice:关闭视频的结果通知
            webUserLocationNotice:点击定位通知

        view listeners:
             注册方法：websdk.listeners.dynamicNotice(msg_code, callback, key);
             notice_im_modal_open:IM窗口打开通知
             notice_im_modal_close:IM窗口关闭通知
             notice_video_modal_open:视频窗口打开通知
             notice_video_modal_close:视频窗口关闭通知
             notice_video_call_modal_open:视频通话窗口打开通知
             notice_video_call_modal_close:视频通话窗口关闭通知
             notice_voice_call_modal_open:语音通话窗口打开通知
             notice_voice_call_modal_close:语音通话窗口关闭通知

        样式定制：
            弹出窗口的唯一类名：（可通过类名进行样式覆盖，如z-index，组件的大小，颜色等）
            创建群组窗口：sdk-group-create-modal
            用户/群组窗口：sdk-user-main-modal
            用户列表窗口：sdk-user-list-modal
            视频通话窗口：sdk-video-call-modal
            视频通话确认窗口：sdk-video-confirm-modal
            视频推拉流窗口：sdk-video-main-modal
            语音通话窗口：sdk-voice-call-modal
            语音通话确认窗口：sdk-voice-confirm-modal
            语音半双工振铃窗口：sdk-voice-ptt-confirm-modal
